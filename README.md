#  Medical Chatbot (NLP + FastAPI)

An End-to-End AI-powered medical chatbot that predicts diseases based on user symptoms using NLP and Machine Learning.

---

##  Features
- Predict disease from symptoms
- NLP preprocessing
- TF-IDF vectorization
- Naive Bayes model
- FastAPI REST API

---

## Tech Stack
- Python
- Scikit-learn
- FastAPI
- NLP

---

##  Run Locally

```bash
git clone https://github.com/YOUR_USERNAME/medical-chatbot.git
cd medical-chatbot
pip install -r requirements.txt
uvicorn app.main:app --reload
